# Linux From Scratch

This is my documented journey building Linux From Scratch 13.0 (systemd). I'll be working through the LFS book step by step compiling a cross-toolchain, building temporary tools, chrooting in, and finally installing a base system that boots on its own.

This repo is where I'm keeping everything I don't want to lose along the way: configs, scripts, patches, build logs, and whatever notes I scribble down when something breaks (or finally works).

I'm following the [LFS book](https://www.linuxfromscratch.org/lfs/view/stable-systemd/). Wish me luck.
